## Requirement

Node >= 4.0.0

## Usage

Install dependencies:
```sh
$ npm install
```

Start the JSON API server:
```sh
$ npm run api-server
```

Open another terminal and running below command:
```sh
$ npm start
```

Open `http://localhost:3000` in your browser.
